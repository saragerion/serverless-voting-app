# serverless-voting-app

This is a demo voting app built with serverless services.

### Technologies

- AWS services: Amazon CloudFront, Amazon API Gateway, AWS Lambda, Amazon DynamoDB, Amazon S3.
- Terraform
- Bash / Github Actions
- K6
